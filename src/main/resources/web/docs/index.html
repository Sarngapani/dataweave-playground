dw::Core

Core DataWeave functions for data transformations.

map

Iterates over an array and applies a transformation to each element.

Signature 
map ((item, index) -> Any)
Example 
payload map $.name

mapObject

Iterates over object key-value pairs and returns a new object.

Signature 
mapObject ((value, key) -> Any)
Example 
payload mapObject ((v, k) -> upper(v))

filter

Filters an array based on a condition.

Signature 
filter ((item) -> Boolean)
Example 
payload filter $.age > 18

match

Pattern matching for values.

Signature 
match { case ... }
Example 
payload match {
  case 1 -> "one"
  case _ -> "other"
}

dw::Strings

String manipulation functions.

upper

Converts a string to uppercase.

Signature 
upper(String): String
Example 
upper("hello")

lower

Converts a string to lowercase.

Signature 
lower(String): String
Example 
lower("HELLO")
